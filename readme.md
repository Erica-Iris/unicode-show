# Unicode Shower

## Description

The function of this extension is to select a piece of unicode code and then display the corresponding character.

## Support Format

 - UTF-8
 - UTF-16
 - UTF-32
 - UTF-7
 - CESU-8
 - GB18030

## Usage

Select the unicode code, then press the shortcut key `ctrl+alt+u`, use `cmd+alt+u` on mac
